# Blazor To-Do App

A simple Blazor Server project demonstrating component usage, form handling, and basic task tracking.

## Project Overview

This project contains a minimal task management system built using Blazor Server. It demonstrates how different components interact while keeping the overall structure simple and easy to follow.

### Key Features

- Dashboard page that displays:
  - Daily Tasks section
  - Monthly Tasks section

- Reusable ToDo component with:
  - Support for a custom header through a component parameter
  - Form for adding tasks
  - Click-to-toggle task completion state
  - Progress bar showing the percentage of completed tasks
  - Visual separation of completed and pending tasks
  - Table displaying only the pending tasks

### Technical Highlights

- Built entirely with Blazor Server
- Uses basic component communication through parameters
- Demonstrates state handling using in-memory lists
- Implements simple UI logic using Bootstrap styling
